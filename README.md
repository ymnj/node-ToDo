# Plans

---

A To Do list application using NodeJS, ExpressJS, MongoDB and Vue.js

##### Locally
```
1. git clone git@github.com:ymnj/plans.git
2. npm install
3. npm run watch-app
4. Open browser at localhost:3000
```

#### Test
```
Testing done with Mocha and Expect

1. npm run watch-test

```
